Taco Cat: A Coding Project

A JavaScript codeing exercise using "arrays" and "for loops" that can determine if a word is a palindrome.
A palindrome is a word or phrase that reads the same backward as forward.
Just type your word and press Palindrome Me! to see the magic happen.
